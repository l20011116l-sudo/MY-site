关于本站

本站为静态 HTML，无构建步骤，便于在任何静态托管商上发布。视觉基调：清水混凝土式的灰阶、直墙般的分割与一侧窄「光缝」，对应安藤作品中常见的材质与采光逻辑（仅为网页上的抽象引用，非建筑学表述）。

如何让别人通过网址打开

将 personal-site 文件夹推送到 GitHub 等仓库，在仓库设置里开启 GitHub Pages（或改用 Vercel / Netlify / Cloudflare Pages 连接同一仓库），会得到形如 用户名.github.io/仓库名 的公开地址。把该链接发给朋友即可。

修改署名与文案

全站品牌名出现在各页顶栏与页脚，可按需替换「蒋诚林」为你的笔名或英文名；首页 hero 区可改写为你的自我介绍。
