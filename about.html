About · Chenglin

关于

我是蒋诚林（Chenglin）。这里记录技术笔记、法律方向的学习与 AI 辅助工作流（例如用 Cursor 做材料外置与备考），也放个人静态站与作品链接。

联系

欢迎通过 GitHub 交流；新文章见 笔记，更新订阅可使用 RSS（Atom）。首页「树屋」视觉入口为 index.html。

关于本站

本站为静态 HTML/CSS，无构建步骤，便于在 GitHub Pages、Vercel、Cloudflare Pages 等任意静态托管上发布。视觉基调：清水混凝土式的灰阶、直墙般的分割与一侧窄「光缝」（仅为网页上的抽象引用，非建筑学表述）。

如何让别人通过网址打开

将仓库推送到 GitHub 后，在仓库 Settings → Pages 中开启 GitHub Pages（源选 main、根目录），会得到形如 用户名.github.io/仓库名 的公开地址。把该链接发给朋友即可。

修改署名与文案

全站品牌名出现在各页顶栏与页脚，可按需替换「蒋诚林」为你的笔名或英文名；首页 hero 区可改写自我介绍与外链。
